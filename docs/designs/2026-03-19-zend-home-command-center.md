# Zend Home Command Center

Generated from the accepted CEO-review expansions on 2026-03-19.

## Vision

Zend should feel like the private operating system for a household mining node.
The user should pair one trusted phone to one named home box, see its live
status, receive private operational receipts and agent summaries in one place,
and never wonder whether the phone is secretly doing the mining.

## Accepted Scope

- appliance-style Zend Home onboarding
- trust ceremony with explicit observe or control grant
- Zend-native gateway contract
- Hermes adapter on top of the Zend contract
- encrypted operations inbox
- append-only private event spine
- shared `PrincipalId` across gateway and future inbox

## First-Slice Storyboard

1. User boots Zend Home and names the box.
2. Phone pairs locally by QR or short code.
3. Trust screen explains what this device can and cannot do.
4. Dashboard shows live miner status and safe operating modes.
5. Operations inbox shows pairing approval, control receipts, alerts, and
   Hermes summaries in one private surface.
6. Off-device audit proves the phone is only a command center.

## Deferred

- remote access beyond the local network
- payout-target mutation
- full conversation UX beyond the operations inbox
- multi-device sync
